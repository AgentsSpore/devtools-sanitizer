# DevTools Sanitizer

A web application for automatically sanitizing sensitive data from development logs and API payloads.

## Features

- Upload log files for auto-sanitization
- Define custom patterns to match sensitive data
- Preview and download cleaned outputs
- Modern, responsive UI

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/devtools-sanitizer.git
cd devtools-sanitizer
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Upload your log file using the drag-and-drop zone or file picker
2. Define patterns for sensitive data detection (or use default ones)
3. Preview the sanitized output
4. Download the cleaned file
